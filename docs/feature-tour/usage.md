# Usage
To use Cloner, navigate to any of the below supported 'grouping' screens in your control panel. You should see a small `+` icon. Click this to open up a prompt window to enter the new name for your group.

Supports cloning:
- Asset Transforms
- Category Groups
- Entry Types
- Filesystems
- Global Sets
- Sections
- Tag Groups
- User Groups
- Volumes
